#README

My attempt to creating a MongoDB, Express, Node server library...

Installation Cmds
-

- `npm install`
  - Requires minimum Node v0.12  

Gulp Cmds
-

- `gulp lint` 
  - Execute linting

- `gulp test` 
  - Execute unit tests

- `gulp serve`
  - Launch server with watchers on .js file changes (also runs lint and test tasks)
